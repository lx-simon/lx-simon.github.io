改用自己的域名

流程：

  1. 进入域名解析台，添加记录,CNAME(主机记录相当于域名的前缀，如www >> www.chenduo.xyz)
  2. 配置GitHub page绑定解析的域名
    (tips：仓库往下划就能找到page)

  3. 这个时候还是有点小问题的
    比如说我只能http访问，浏览器显示不安全，有些甚至不能访问
    我们去解析太在刚刚创建CNAME边上有SSL免费申请
    然后 ，GitHub这边就能直接打开使用https了
